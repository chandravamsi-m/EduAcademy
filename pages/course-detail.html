What You'll Learn

Build responsive, mobile-first websites from absolute scratch.

Course Summary

This curriculum is built from scratch for aspiring computer engineers and layout design enthusiasts. Over 12 structured modules, students learn how semantic markup, absolute layouts, flexbox coordinates, form validation, and responsive stylesheets work together to build stunning user applications.

Curriculum & Syllabus

Study standard structures: headers, footers, main containers, articles, navigation boxes, and search overlays. Keyboard accessibility and screen reader rules.

  • Understanding semantic tags vs generic divs
  • Creating accessible documents with ARIA landmarks
  • Structuring complex tables and interactive elements
3 Lessons · 1h 45m

Master box model physics, centering strategies, and responsive alignments using Flexbox flex-basis and CSS Grid template columns.

  • Block vs inline layouts and display rules
  • Flexbox container axes, alignment, and gap options
  • Building complex grids with grid-template-areas
4 Lessons · 2h 15m

Learn utility classes vs custom CSS variables. Color systems, typography properties, custom themes, dark mode variants, and responsive media breakpoint settings.

  • Mobile-first design workflow and design patterns
  • Configuring tailwind.config.js for custom brand colors
  • State modifiers: hover, focus, active, dark-mode
4 Lessons · 2h 30m

Implement functional forms with validation, accessible label pairing, and interactive status feedback styling.

  • Input types, labels, and textareas styling
  • HTML5 native form validators and custom error classes
  • Focus indicators and keyboard navigation loops
3 Lessons · 1h 50m

Explore CSS custom properties, calc functions, transitions, and hover effects that make interfaces look premium.

  • CSS Variables runtime adjustments
  • Transitions and transform micro-animations
  • Glassmorphism design parameters
3 Lessons · 2h 05m

Integrate everything learned to create a full portfolio site, complete with landing hero, work grids, filter logic, and contact elements.

  • Project setup and folder architecture
  • Structuring pages and cross-navigation links
  • Hosting deployment workflows (GitHub Pages/Vercel)
7 Lessons · 4h 30m

Requirements & Prerequisites

  • No prior coding or web design experience required. We start completely from scratch.
  • A working computer (Windows, macOS, or Linux) with connection to the internet.
  • We will install VS Code and relevant extensions together during the first modules.
